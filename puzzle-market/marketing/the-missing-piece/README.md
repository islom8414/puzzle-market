# The Missing Piece

Production package for a short vertical AI animated ad for Puzzle Market.

Brand: Puzzle Market  
Website: https://www.puzzle-market.com  
Hero character: Puzz  
Format: 9:16 vertical, 20-25 seconds, 1080p or higher

Files:

- `runway-master-prompt.md` - full prompt for a single video generation pass.
- `scene-prompts.md` - scene-by-scene prompts for multi-shot generation.
- `captions-and-voiceover.md` - English text overlays and optional voiceover.
- `publishing.md` - title, description, hashtags, pinned comment, upload notes.
- `cover-prompt.md` - prompt for the Shorts/Reels cover image.

Creative note: use a premium cinematic 3D family animation look. Avoid directly naming or copying any specific studio style in final generated metadata.
